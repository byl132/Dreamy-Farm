# 🌾 梦境农场 - Dreamy Farm

二次元风格农场经营游戏，使用 Godot 4.x 开发。

> *"在梦境的田野里，种下希望，收获奇迹。"*

---

## 🎮 当前版本功能

### ✅ 已完成
- **玩家控制** - WASD / 方向键移动
- **角色朝向** - 自动翻转面向移动方向
- **摄像机跟随** - 平滑跟随玩家
- **农田系统** - 6x4 网格农田
- **种植机制** - 5种作物，随时种植
- **生长系统** - 实时生长进度显示
- **收获系统** - 点击收获，自动获得金币
- **时间系统** - 日夜循环、季节变化
- **季节系统** - 春夏秋冬影响种植
- **任务系统** - 4个引导任务
- **存档系统** - 自动保存游戏进度
- **UI界面** - 原神风格美化的界面

## 🎮 操作指南

| 按键 | 功能 |
|------|------|
| WASD / ↑↓←→ | 移动角色 |
| 鼠标左键 | 与农田交互（种植/收获） |
| 数字键 1-5 | 切换种子类型 |
| ESC | 关闭/暂停 |

## 📂 项目结构

```
dreamy_farm/
├── assets/              # 资源文件
│   ├── audio/           # 音效
│   ├── fonts/           # 字体
│   ├── sprites/         # 精灵图
│   └── tilesets/        # 瓦片集
├── scenes/              # 场景
│   ├── ui/             # UI场景
│   │   ├── start_menu.tscn
│   │   └── game_hud.gd
│   └── world/          # 游戏世界
│       └── main_world.tscn
├── scripts/             # 脚本
│   ├── core/           # 核心系统
│   │   ├── game_manager.gd    # 游戏状态
│   │   ├── audio_manager.gd   # 音频管理
│   │   └── save_system.gd     # 存档系统
│   ├── systems/        # 游戏系统
│   │   ├── player.gd         # 玩家控制
│   │   ├── farm_system.gd    # 农田系统
│   │   ├── world.gd          # 世界管理
│   │   ├── npc_system.gd     # NPC系统
│   │   └── quest_system.gd   # 任务系统
│   └── ui/             # UI脚本
│       ├── game_hud.gd
│       └── start_menu.gd
├── resources/           # 数据
│   └── saves/          # 存档目录
├── GDD.md              # 游戏设计文档
├── project.godot       # 项目配置
└── README.md
```
