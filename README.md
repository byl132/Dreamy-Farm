# 🌾 梦境农场 - Dreamy Farm

二次元风格农场经营游戏，使用 Godot 4.x 开发。

> *"在梦境的田野里，种下希望，收获奇迹。"*

---

## 🎮 当前版本功能（V0.2.3）

### ✅ 已完成
- **玩家控制** - WASD / 方向键移动
- **角色朝向** - 自动翻转面向移动方向
- **摄像机跟随** - 平滑跟随玩家
- **农田系统** - 6x4 网格农田
- **种植机制** - 5种作物，随时种植
- **生长系统** - 实时生长进度显示
- **收获系统** - 点击收获，自动获得金币
- **时间系统** - 日夜循环、季节变化
- **季节系统** - 春夏秋冬影响种植
- **任务系统** - 4个引导任务
- **存档系统** - 自动保存游戏进度
- **UI界面** - 原神风格美化的界面

---

## 🎮 操作指南

| 按键 | 功能 |
|------|------|
| WASD / ↑↓←→ | 移动角色 |
| 鼠标左键 | 与农田交互（种植/收获） |
| 数字键 1-5 | 切换种子类型 |
| ESC | 关闭/暂停 |

## 1. 核心系统设计

### 1.1 种植系统

#### 1.1.1 作物种类
| 作物名称 | 生长周期 | 基础收益 | 最佳季节 | 特殊效果 |
|---------|---------|---------|---------|----------|
| 萝卜 | 2天 | 10金币 | 春季 | 无 |
| 番茄 | 3天 | 20金币 | 夏季 | 无 |
| 草莓 | 4天 | 30金币 | 春季 | 收获时有几率获得种子 |
| 玉米 | 5天 | 40金币 | 夏季 | 无 |
| 南瓜 | 6天 | 50金币 | 秋季 | 无 |
| 郁金香 | 3天 | 25金币 | 春季 | 提高周围作物生长速度 |
| 向日葵 | 4天 | 35金币 | 夏季 | 吸引蜜蜂，提高授粉率 |

#### 1.1.2 种植流程
1. 使用锄头开垦土地
2. 选择种子进行种植
3. 定期浇水（使用 watering can）
4. 等待作物生长
5. 收获作物获得收益

#### 1.1.3 影响因素
- **天气**：雨天自动浇水，干旱需要更多浇水
- **季节**：不同季节适合种植不同作物
- **温湿度**：温湿度适宜度影响生长速度和产量
- **土壤肥力**：可使用肥料提高土壤肥力

### 1.2 养殖系统

#### 1.2.1 动物种类
| 动物名称 | 购买价格 | 每日消耗 | 产出 | 产出周期 |
|---------|---------|---------|------|----------|
| 鸡 | 50金币 | 5金币 | 鸡蛋 | 1天 |
| 牛 | 200金币 | 15金币 | 牛奶 | 1天 |
| 羊 | 150金币 | 10金币 | 羊毛 | 2天 |
| 猪 | 250金币 | 20金币 | 猪肉 | 3天 |
| 马 | 500金币 | 25金币 | 无 | 无（可骑乘） |

#### 1.2.2 养殖流程
1. 购买动物
2. 建造相应的畜棚
3. 定期喂养动物
4. 收集动物产品
5. 出售产品获得收益

#### 1.2.3 影响因素
- **饲料**：不同饲料影响动物产出质量
- **居住环境**：畜棚等级影响动物健康度
- **温湿度**：适宜的温湿度提高动物产出
- **卫生状况**：定期清洁畜棚，避免疾病

### 1.3 建筑系统

#### 1.3.1 建筑种类
| 建筑名称 | 建造价格 | 升级价格 | 功能 | 二级场景 |
|---------|---------|---------|------|----------|
| 温室 | 500金币 | 300金币 | 不受季节影响种植 | 是 |
| 仓库 | 300金币 | 200金币 | 存储物品 | 是 |
| 加工厂 | 800金币 | 500金币 | 加工农产品 | 是 |
| 鸡舍 | 200金币 | 150金币 | 养鸡 | 是 |
| 牛舍 | 400金币 | 300金币 | 养牛和羊 | 是 |
| 工具棚 | 150金币 | 100金币 | 存放工具 | 是 |
| 商店 | 600金币 | 400金币 | 购买物品 | 是 |

#### 1.3.2 建筑升级
- 升级建筑可提高功能和容量
- 升级需要消耗金币和材料
- 高级建筑解锁新功能

### 1.4 NPC系统

#### 1.4.1 NPC种类
| NPC名称 | 角色 | 功能 | 出现时间 |
|---------|------|------|----------|
| 樱花 | 商店老板 | 出售种子和物品 | 9:00-18:00 |
| 杰克 | 农民 | 提供种植建议 | 7:00-12:00, 14:00-18:00 |
| 玛丽 | 动物管理员 | 提供养殖建议 | 8:00-17:00 |

#### 1.4.2 NPC行为
- NPC有自己的日常作息时间
- NPC会根据天气和季节调整行为
- NPC有情绪系统，受玩家互动影响
- NPC会提供任务和奖励

### 1.5 温湿度系统

#### 1.5.1 温湿度影响
- **作物**：适宜温湿度提高生长速度和产量
- **动物**：适宜温湿度提高产出质量
- **玩家**：极端温湿度影响玩家行动

#### 1.5.2 调节方法
- **建筑**：不同建筑有不同的温湿度调节能力
- **设备**：可购买温湿度调节设备
- **自然因素**：天气和季节影响温湿度

## 2. 游戏界面设计

### 2.1 主界面
- 农场主视图：显示农场全景
- 工具栏：快捷访问工具和物品
- 状态栏：显示玩家状态和资源
- 菜单按钮：访问设置、商店等功能

### 2.2 二级场景
- 建筑物内部：可进行特定活动
- 菜单界面：设置、商店、背包等
- 交易界面：与NPC交易
- 任务界面：查看和管理任务

### 2.3 UI元素
- 图标：使用简洁明了的图标
- 颜色：使用温暖的自然色调
- 字体：清晰易读的字体
- 动画：流畅的过渡动画

## 3. 数值平衡设计

### 3.1 经济平衡
- 初期收入：以种植和养殖为主
- 中期收入：加工和交易
- 后期收入：高级作物和动物
- 价格波动：根据市场供需调整

### 3.2 时间平衡
- 生长周期：合理的时间长度
- 任务时间：适合休闲玩家的时间长度
- 活动频率：定期更新的活动和事件

### 3.3 难度平衡
- 初期：简单易上手
- 中期：逐渐增加挑战
- 后期：深度玩法和策略
- 可选难度：适合不同类型的玩家

## 4. 游戏流程设计

### 4.1 新手引导
1. 教程任务：引导玩家熟悉基本操作
2. 奖励系统：完成教程获得奖励
3. 渐进式解锁：逐步解锁新功能

### 4.2 中期玩法
1. 扩展农场：扩大农场规模
2. 解锁建筑：建造各种功能性建筑
3. 完成任务：与NPC互动完成任务
4. 探索区域：探索农场周围的区域

### 4.3 后期玩法
1. 优化农场：提高农场效率和收益
2. 收集成就：完成各种游戏成就
3. 参与活动：参加季节性活动
4. 社交互动：与其他玩家互动
